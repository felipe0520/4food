### Principais tecnologias utilizadas:

4. Axios
5. Redux-Thunk
6. Material-ui


## Como rodar a aplicação 


No terminal, clone o projeto:

```git clone: https://github.com/felipe0520/4eddit ```

Entre na pasta do projeto  :
```cd 4eddit ```

Instale as dependências : 
```npm install ```

Execute a aplicação:
```npm start ```
